How Hydrogen Works

A look under the hood of our intelligent coding agent

1

Codebase Scanning & Context Building

Hydrogen begins by recursively scanning your project directory, reading file contents, and building a complete understanding of your codebase structure. This deep context is included in every request to the AI model.

2

Task Analysis & Submode Activation

The AI checks if the task contains special directives like @scaffold or @code, activating specialized workflows for project planning (@scaffold) or implementation (@code). Each submode has optimized behavior for its purpose.

3

Action Execution (With Approval)

In Agent Mode, Hydrogen suggests concrete actions like file creation/editing, asks for your confirmation, then executes them. You maintain full control while benefiting from AI-powered automation.
